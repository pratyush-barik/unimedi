# UniMedi

## Disclaimer

UniMedi is currently under active development. Features, workflows, APIs, and user interface components may change as development progresses. This repository represents the current implementation of the platform and additional functionality will be introduced in future releases.

**Live Demo:** https://unimedi.vercel.app/

---

## Overview

UniMedi is a healthcare management platform designed to simplify interactions between patients and healthcare professionals through a centralized digital system. The platform currently provides secure authentication, role-based dashboards, and QR-based doctor-patient connectivity.

The project aims to create a scalable healthcare ecosystem that improves accessibility, streamlines communication, and enables efficient healthcare management.

## System Workflow

1. Users create an account and authenticate securely.
2. Doctors generate unique QR codes.
3. Patients scan a doctor's QR code to establish a connection.
4. Connected users can access role-specific dashboards.
5. Future updates will expand functionality with healthcare record management and additional services.
